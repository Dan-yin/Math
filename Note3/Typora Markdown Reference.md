## 文章和段落

`Enter`  用于创建段落，使用1行空白；`shift`+`Enter` 用于创建换行，无空白行 ;`<br/>`使用两空白行间隔

`Enter`在标题后使用会自动创建新标题;`>`用于创建引用

`*,+,-`,`1.`自动生成列表,`- [ ]`用于生成 任务列表

` ``` ` +`Enter` 生成代码块,` ``` ` + `语言（python）` 生成指定语言代码块

 ` $$` +`Enter`生成latex数学公式，`|姓名|年龄|`生成表格

`[^ 注释]` 可以用于生成注释，点击[^注释]可以快速生成注释



[^注释]: input description here

`***`和`---`在空白行会产生分割线

`[TOC]`会生成目录
